Editing

Google WebFonts

  1. Browse the assets > app > css folders, and open app.css file in any text editor (Notepad, Notepad++, Dreamweaver, etc.).
  2. Replace/ add the following code with new google webfont URL that your select: 
    app.css - line 37/*  Google WebFonts  */
@import url(http://fonts.googleapis.com/css?family=Open+Sans:300,300italic,400,400italic,600,600italic,700,700italic);
  3. Next, you can replace font-family on body (and other) 
    app.css - line 53body {
  font-family: 'Open Sans', sans-serif;
}

Search Suggestions

Search suggestion only work on seach form in header area.
  1. Open the search-term.php file in any text editor (Notepad, Notepad++, Dreamweaver, etc.).
  2. Add your term suggestion by add the following code: 
    $search_terms[] = '';	// TYPE VALUE BETWEEN SINGLE QUOTE
Important! Please don't remove any code in this file, exclude $search_terms[] = '';.

Google Map

  1. Open the contact.html file in any text editor (Notepad, Notepad++, Dreamweaver, etc.).
  2. Found the following script code at line 179 - 200: 
    <script>
  var map;
  function initialize(){
    var mapOptions = {
      zoom: 16,							// MAP ZOOM
      center: new google.maps.LatLng(-7.95860,112.63196),	// MAP COORDINATES
      mapTypeId: google.maps.MapTypeId.ROADMAP
    };
    var map = new google.maps.Map(document.getElementById("map-maker"), mapOptions);
    var marker = new google.maps.Marker({
      position: map.getCenter(),
      map: map,
      title: "NowKnow Office"					// MAP TITLE
    });
    google.maps.event.addListener(map, "center_changed", function(){
      window.setTimeout(function(){
        map.panTo(marker.getPosition());
      }, 3000);
    });
  }
  google.maps.event.addDomListener(window, "load", initialize);
</script>
  3. Replace the highlight code with your map data.
    Map coordinates format is: latitude, longitude.
You can find the your map latitude and longitude coordinates here.
